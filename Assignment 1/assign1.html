DEVELOPING A BASIC WEB PAGE

OBJECTIVES

In this tutorial you will:
  • Explore the structure of the World Wide Web
  • Learn the basic principles of Web documents
  • Create an HTML document
  • View an HTML document using a Web browser
  • Use HTML tags for text, headings, paragraphs, and lists
  • Insert an inline graphic image into an HTML document
  • Insert a horizontal line into an HTML document

SUMMARY

In this tutorial you will learn the basics of the World Wide Web structures and HTML language. You will be able to create a simple Web page. You will learn how to create and apply HTML tags to format page headings, paragraphs, and individual characters. You will also learn how to insert an inline image and a horizontal line.
